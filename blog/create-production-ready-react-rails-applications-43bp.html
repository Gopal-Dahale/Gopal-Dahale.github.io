This is a base project to quickly spin up a React+Rails application.

If you want to create a react application with rails backend then most basic and important thing is authentication. And with this api template you don't need to worry about authentication. In rails app, token based authentication is implemented by using devise gem.

Built with

  • Front-end library - Reactjs
  • Back-end - Rails
  • UI components - Chakra UI
  • Data management - Context Api
  • Authentication - Devise gem
  • Front-end validations - Formik
  • Animation library - Framer Motion
